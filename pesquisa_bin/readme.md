Esse repositorio é um estudo de busca binaria, apresentado pelo livro "Entendendo Algotimos".

O arquivo @variaveis.py, contem a função que cria uma lista de numeros ordenados sendo X o range de numeros (0,x) e Y a quantidade de posições dentro do intervalo.

Em @pesquisa_de_lista.py, encontrase o estudo de caso dito, função que recebe o a lista gerada e identifica se encontra o valor fornecido com pesquisa binaria. 

@sortear_lista foi uma abstração do estudo incial. Essa função que demandou a criação da função de criar listas com X e Y. 
Ela verifica se o numero buscado está na lista e se não, gera uma nova lista até o numero ser encontrado. Com esse projeto incial, vou montar um script que identifique quantas tentativas são necessarias para acertar um conjunto de numeros dentro de uma lista com valores aleatórios.
